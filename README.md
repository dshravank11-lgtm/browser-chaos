# Browser Chaos 67

Turn your browser into a chaotic playground with this extension. 

## Features of Browser Chaos 

### Rickroll

Whenever you open a new tab,
There is a 10% chance you are redirected to a page where you will get rickrolled. The page is not made by me.

### The Secret Code

If you type 67 on any webpage the whole page will flip down. It is a cool feature of Browser Chaos 67.

### The Skull Effect

Here are a few things that the Skull Effect does in Browser Chaos 67:

The skull will be at the bottom of the page and will appear above everything except ads
Music is played at random from 4 possible tracks
it makes the page applying a grayscale filter, like an edit on YouTube or Tiktok

### Instant Skull Mode

This is what happens when you use Instant Skull Mode in Browser Chaos 67:

Your screen will freeze for 3 seconds
Then, a skull will. Shake around.
It will play some music loud.

### Mining Mode

To activate Mining Mode in Browser Chaos 67 you just need to type minecraft.

It will create a grid of blocks that you can break.
You can click on each block to break it. inspired by google's Minecraft easter egg
It is still in progress and isnt good yet.
I have some plans to improve it like adding a moving hand and holding instead of clicking

### Auto Timer

You can set it to trigger the skull effects at times.
You can choose how often it happens. the default interval is 5 minutes
