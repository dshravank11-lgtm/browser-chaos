
# Browser Chaos 

Turn your browser into a chaotic playground with this extension. 

## Features of Browser Chaos 

### New Tab Rickroll

Whenever you open a new tab,
There is a 10% chance you are redirected to a page where you will get rickrolled. The page is not made by me.

### The Secret Code

If you type 67 on any webpage the whole page will flip down. It is a cool feature of Browser Chaos.

### The Skull Effect

Here are a few things that the Skull Effect does in Browser Chaos :

The skull will be at the bottom of the page and will appear above everything except ads
Music is played at random from 4 possible tracks
it makes the page applying a grayscale filter, like an edit on YouTube or Tiktok

<img width="2526" height="1376" alt="image" src="https://github.com/user-attachments/assets/7efe7d9e-e1d3-4fab-be74-f5c2901cd51c" />


### Instant Skull Mode

This is what happens when you use Instant Skull Mode in Browser Chaos 67:

Your screen will freeze for 3 seconds
Then, a skull will appear with some music in the background for 5 seconds.

### Mining Mode

To activate Mining Mode in Browser Chaos you just need to type "mine".

It will create a 4x4 grid of blocks that you can break by clicking.
You have to click on each block 3 times to break it. I'm still working on the sound effects and and textures.
<img width="2555" height="1385" alt="image" src="https://github.com/user-attachments/assets/df2a2c1f-0e11-498e-aefc-24e36d7eb5e9" />


### Auto Skull Timer

You can set it to trigger the skull effects at times.
You can choose how often it happens. the default interval is 5 minutes

# Levitation and Gravity Effect

If "fall" is typed, everything, or at least most things on the screen, will fall to the bottom at an angle.
Likewise, if "leviosa" like wingardium leviosa is typed, then, everything on the screen will start floating up.

# Random Tab Names

Every 30 seconds, there is a 30% chance that the title of a tab in the background can turn into something weird like 'free robux click here'
